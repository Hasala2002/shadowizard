# What is this?

Get perfect shadows in CSS

# Installation

`npm i shadowmaster --save`

Then...

```
import {shadowmaster} from 'shadowmaster';

shadowmaster({
    shadow_type: 'soft',
    padding: 'false'
});
```

# Options

* *shadow_type* - _hard | soft_ (Defaults to soft)
* *padding* - _boolean_ (Defaults to false)
